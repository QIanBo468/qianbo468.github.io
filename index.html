搭建个人博客

Posted on 2019-07-25 Edited on 2021-03-18

什么是 Hexo？

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

https://hexo.io/zh-cn/
这是他的文档

安装前提

怎么安装 git 和 node.js 就不多讲了，谷歌，百度上都有详细教程，这里主要说明一个怎么查看是否安装成功的方法

  • Node
    • node -v 如果返回版本号，则说明成功
  • Git
    • git version 同上

进行安装

  1. 安装 hexo-cli

    npm install -g hexo-cli

  2. 初始化

    hexo init blog

    后续的操作都需要在此文件夹下

  3. 切换至 blog 文件夹
    npm install 安装依赖

  4. hexo server 本地启动

文件描述

  • _config.yml

    配置文件, 可在里面修改一些页面属性

  • package-lock.json

    所使用到到的第三方插件

  • themes

    主题

  • scaffolds

    生成的一些文章模版和一些原属性

  • package.json

    自己安装的库

  • public

    生成的博客

  • source

    创建的文章

https://hexo.io/themes/

这里面的每一个主题都很精美,
进入到主题的 github 中,将它克隆到你本地的 theme 文件夹中

在配置文件(_config.yml)中 修改你所选择的主题

我这里用的是 next

最后保存,重新启动就行了

创建文章

hexo new name

会在_post 文件夹下新建一个 你 name 的 md 文件
